# Slack Clone

A Slack clone built with Node.js, Express, and Socket.io

## About

This project is a Slack clone built with Node.js, Express, and Socket.io. The goal of this project is to create a real-time messaging application that has the same features as Slack.

## Features

- Real-time messaging
- Room message support
- User authentication
- User typing indicators
- Multi-room support
- User information display (name, profile picture)

## Technologies Used

- Node.js
- Express
- Socket.io

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository
2. Install the dependencies
3. Start the server
4. Open the application in a web browser

To test this application, you will need to open two browsers. One for each user.
